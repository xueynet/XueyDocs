# <stl:printer> 打印

```html
<stl:printer
    titleId="标题的Id"
    bodyId="正文的Id"
    logoId="Logo的Id"
    locationId="当前位置的Id">
</stl:printer>
```

### 说明

`stl:printer` 用于显示打印链接，点击后弹出打印窗口

`stl:printer` 标签将显示打印链接，点击此链接后将弹出打印窗口，需打印的标题、内容、LOGO及页面当前文字在 `<stl:printer>` 的属性中指定。

`<stl:printer>` 标签对应的实体为`{stl:printer}`。

### 属性

::: tip
所有属于HTML 标签`<a>`的属性均适用于`<stl:printer>`标签。
:::
